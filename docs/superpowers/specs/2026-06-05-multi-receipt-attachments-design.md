# Task 22 사전 첨부파일 다중 업로드와 월별 다운로드 설계

작성일: 2026-06-05, Asia/Seoul

## 목표

Task 22 스테이징 사용자 시나리오 점검 전에 영수증 첨부 기능을 실제 사용 방식에 맞게 확장한다. 한 신청서에 여러 장의 영수증 이미지를 올릴 수 있게 하고, 신청서 상세에서 이미지를 편하게 확인하며, 관리자는 월 단위로 첨부 이미지를 ZIP 묶음으로 다운로드할 수 있게 한다.

## 범위

- 신청서당 첨부 최대 개수를 기본 10개로 늘린다.
- 파일당 기본 크기 제한은 5MB로 유지한다.
- 신청서 작성/수정 화면에서 여러 이미지 파일을 한 번에 선택할 수 있게 한다.
- 선택한 이미지들은 썸네일 그리드, 개별 삭제, 확대 미리보기를 제공한다.
- 제출/임시저장 시 선택한 여러 이미지를 순차 업로드한다.
- 신청서 상세 화면은 여러 첨부 이미지를 보기 쉬운 썸네일 그리드와 확대 모달로 제공한다.
- 관리자 화면에서 기준 월을 선택해 해당 월 영수증 첨부 이미지를 ZIP으로 다운로드한다.
- 월별 ZIP 기준은 신청서의 `receipt_date` 월이다.

## 비범위

- 첨부파일 PDF, 문서, 엑셀 업로드는 포함하지 않는다.
- 기존에 업로드된 첨부파일의 개별 삭제 API는 포함하지 않는다.
- ZIP 다운로드를 일반 신청자나 결재자에게 열지 않는다.
- 외부 스토리지 연동은 포함하지 않는다.

## 설계

### 다중 첨부 정책

서버 기본값은 `ATTACHMENT_MAX_FILES_PER_APPLICATION=10`으로 변경한다. `ApplicationService`는 기존과 같이 저장 전에 현재 첨부 개수를 확인하되, 새 최대값까지 허용한다. API는 파일당 크기 제한을 유지하며, MIME 타입과 image signature 검증은 그대로 적용한다.

### 신청서 작성 UI

파일 input은 `multiple`을 사용한다. 사용자가 여러 이미지를 선택하면 프론트에서 형식과 크기를 먼저 검증한 뒤 목록에 추가한다. 목록은 파일명, 크기, 썸네일, 확대 미리보기 버튼, 삭제 버튼을 제공한다. 같은 화면에서 기존 첨부 개수와 새로 선택한 파일 수의 합이 최대 10개를 넘으면 차단한다.

### 신청서 상세 UI

상세 화면의 첨부 영역은 여러 이미지를 한눈에 볼 수 있는 그리드로 표시한다. 썸네일을 클릭하면 큰 미리보기 모달이 열리고, 모달 안에서 이전/다음 이미지로 이동할 수 있다. 모바일에서도 버튼과 이미지가 겹치지 않도록 고정 크기 썸네일과 스크롤 가능한 큰 이미지 영역을 사용한다.

### 월별 ZIP 다운로드

관리자 전용 API를 추가한다.

```text
GET /api/admin/attachments/monthly-download?month=YYYY-MM
```

`ADMIN`만 접근 가능하다. `MANAGER`는 전체 신청서 조회 권한이 있어도 ZIP 다운로드는 할 수 없다. ZIP 대상은 `applications.receipt_date`가 해당 월에 속한 첨부파일이다. ZIP 내부 경로는 아래처럼 식별 가능하게 만든다.

```text
YYYY-MM/application-<id>/<receipt-date>-<vendor>-<attachment-id>-<original-filename>
```

파일명에 사용할 수 없는 문자는 안전한 문자로 치환한다. 대상 첨부가 없으면 빈 ZIP을 내려준다.

### 관리자 UI

`/admin/applications` 화면 상단에 월 선택 input과 다운로드 버튼을 추가한다. 기본값은 현재 월이다. 클릭하면 blob 응답을 받아 `receipt-attachments-YYYY-MM.zip` 파일로 저장한다.

## 테스트

- 서비스 테스트: 기본 최대 10개까지 첨부 가능하고 11번째는 차단한다.
- API 테스트: `ADMIN`은 월별 ZIP 다운로드 가능, `MANAGER`는 403이다.
- API 테스트: ZIP에는 해당 월 첨부만 포함되고 다른 월 첨부는 제외된다.
- 프론트 테스트: 신청서 작성 화면에서 여러 파일을 선택하면 여러 첨부 업로드 API가 호출된다.
- 프론트 테스트: 관리자 화면에서 월별 ZIP 다운로드 버튼이 blob 다운로드를 호출한다.
- 프론트 테스트: 상세 화면에서 첨부 이미지를 클릭하면 확대 미리보기가 열린다.

## 결정 사항

- 기본 첨부 최대 개수는 10개로 한다.
- 파일당 기본 크기는 5MB로 유지한다.
- 월별 ZIP 다운로드는 `ADMIN` 전용으로 시작한다.
- 월별 ZIP 기준은 신청일이 아니라 영수증 일자인 `receipt_date`로 한다.
